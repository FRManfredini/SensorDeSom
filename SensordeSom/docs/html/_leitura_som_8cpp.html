Detailed Description

Leitura de ADC na placa DK32MP usando sysfs.

Este arquivo contém a classe LeituraSom, que permite ler valores brutos de um ADC e convertê-los em tensão em Volts.

Exemplo de uso:

LeituraSom adc("/sys/bus/iio/devices/iio:device0/in_voltage13_raw");
if(adc.ler()) {
std::cout << adc.getLeitura() << " | " << adc.getTensao() << " V\n";
}
