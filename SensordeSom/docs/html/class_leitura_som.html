Detailed Description

Classe para leitura de valores do ADC via sysfs.

Esta classe encapsula a leitura de um ADC representado como arquivo no Linux. Permite ler o valor bruto e convertê-lo em tensão. Também é compatível com Doxygen para gerar diagramas de classe e gráficos de chamadas.

Note
Esta classe acessa diretamente arquivos em /sys/bus/iio/devices/
