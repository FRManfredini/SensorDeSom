\doxysubsection{Detailed Description}
Exemplo de leitura de ADC na placa DK32\+MP usando interface sysfs. 

Este código implementa uma classe para acessar o conversor analógico-\/digital (ADC) via arquivos do sistema em Linux. Ele permite obter o valor bruto da leitura e convertê-\/lo para tensão em Volts.
