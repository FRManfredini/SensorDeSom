\doxysubsection{Detailed Description}
Classe para leitura de valores do ADC através da interface sysfs. 

A classe encapsula o acesso a um ADC representado como arquivo no sistema Linux. Fornece métodos para obter a leitura bruta e convertê-\/la em tensão.
